# Peso Felino

Aplicativo desenvolvido como parte do Trabalho de Conclusão de Curso em Sistemas de Informação.  

## Dependencias
- React Native  
- Expo  
- Chart.js  
- AsyncStorage  

## Funcionalidades
- Cadastro de informações do gato (nome, sexo, porte e castração)  
- Registro semanal do peso  
- Exibição de gráfico com evolução do peso  
- Armazenamento local dos dados  

## Execução do Projeto
1. Instale as dependências:
   ```bash
   npm install
1. Execute o Expo:
   ```bash
   npx expo start
